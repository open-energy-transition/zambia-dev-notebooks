# zambia-dev-notebooks
This repo creates a quick and easy way for notebooks created during development to be shared and easily accessed

## List of contributions

- **hydro_generation_validation.ipynb** explores hydro generation in a modelling workflow
- **inflow_validation.ipynb** provides outputs of inflow validation. Supporting data are shared [here](https://drive.google.com/drive/folders/1bkmEHgj2xJyUpHKURp54d_BTzI3gY2pp?usp=drive_link)
